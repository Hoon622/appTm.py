# appTm.py
#### 티치블머신에서 만든 모델은 파이썬 SW로 구성했다. 가상환경은 
```
conda create -n tm006 python=3.7
conda activate tm006
```
#### 가상환경 패키지 다운로드는
```
pip install pillow
pip install tensorflow opencv-python numpy
```
#### 실행은
```
python apptm.py
```
